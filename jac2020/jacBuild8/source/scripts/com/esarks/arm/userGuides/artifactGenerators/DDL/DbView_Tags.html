Element Description
views This is the XML defintion file used by the following scripts:
  • com.esarks.arm.model.MakeJeo
  • com.esarks.arm.model.MakeDdl
Element Description
view A container tag for the generated view.
Attribute Description
name The name of the view.
Attribute Description
where An optional where clause.
Element Description
table A container tag for the table.
Attribute Description
name The table name containing the columns identified below.
Attribute Description
in The package location of the table Jeo.
Attribute Description
symbolic When used assigns a symbolic identifier for the table. Should be used when referencing columns with the same names located in disparate tables.
Element Description
column A container tag for the column.
Attribute Description
name The name of the referenced column.
Attribute Description
nameView The name given to this column in the view.
Attribute Description
isName The value for is name inserts the function hrmConcatName to concatenate the values of lastName, firstName, middleInitial, and suffix into one name.
Attribute Description
view The name given to this column in the view.
Attribute Description
isColumn If the value is "false" than this column is not referencing a table column. Ex. The name would contain a reference to a function.
Attribute Description
type The data type of the contents of the column.
Attribute Description
isType If the value is "true", it inserts the column in the view after referencing the hrmTypeTranslation table which determines the types of values to be associated with the column.
    Ex. by referencing the hrmTypeTranslation table the data type for this column will be 'string' while the possible values will be 'M','F', or 'U' which will be further translated to 'Male', 'Female', and 'Unknown' respectively.
