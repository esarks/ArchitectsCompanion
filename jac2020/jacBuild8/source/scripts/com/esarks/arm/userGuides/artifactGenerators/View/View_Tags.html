Element Description
frames This is the XML defintion file used by the following scripts:

com.esarks.arm.view.GenerateFrame
Element Description
frame The frame to be created.
Attribute Description
name The application reference name of the frame.
Attribute Description
title The name of the frame as displayed in the browser.
Element Description
control Contains the definition of the control.
Attribute Description
name The name of the control.
Attribute Description
type The type of control.

Ex: multirow, data, text, select, checkbox.
Attribute Description
grid The specific grid in which the control is located.
Attribute Description
column The specific column within a grid in which the control is located.
Attribute Description
maxlength Maximum length of the data entered into a text box.
Attribute Description
size Length of the text box in a text control.
Attribute Description
caption Text to be displayed with the control in the browser.
Attribute Description
uppercase When 'true' changes any data from lower case to caps when submitted from the control.
Element Description
option The name of the values for a multirow or checkbox as displayed in the browser.
Attribute Description
value The specific value of an option.

Ex.

Above "03" is the application specific value for the option Black or African-American.
Element Description
button A container tag for the button.
Attribute Description
type The type of action the button will perform upon activation.

Ex. submit,reset, etc.
Attribute Description
name The application reference name of the button.
Attribute Description
value The name of the button as displayed in the browser.
Attribute Description
method The method to be executed upon button activation.
Element Description
service A container tag for the service being called by the button.
Attribute Description
script The name of the script containing the service being called by the button.
Attribute Description
method The name of the service being used by the button.
Attribute Description
map The specific map to use to display in the browser.
Element Description
maps A container tag for the maps for a view.
Element Description
map A specific map for the view.
Attribute Description
name The name of the specific map.
Element Description
jeo A container tag for the Jeo for use with a map or maps.
Attribute Description
name The name of the Jeo.
Attribute Description
control Contains a refrence to a control in order to set the context for the mapping.
Element Description
property A container tag for the property within a Jeo.
Attribute Description
from The name of the property containing a value for display in a specific part of the view.
Attribute Description
to The specific control recieving data from the from tag.
Attribute Description
value The specific value of a given property.
Attribute Description
rowColumn A specific placement in a multirow control.
